# 🐍 30 Days Python Coding Challenge

This repository contains my journey of completing a **30 Days Python Coding Challenge**. The goal of this challenge is to strengthen my understanding of Python programming by solving coding problems daily and improving logical thinking and problem-solving skills.

## 🎯 Objectives

- Build a strong foundation in Python programming
- Practice daily coding to improve consistency
- Enhance problem-solving and logical thinking
- Learn and apply Python concepts through practical exercises
- Maintain a public coding portfolio on GitHub

## 📚 What This Repository Contains

Each day includes a Python script that focuses on a specific concept such as:

- Basic Python syntax
- Variables and data types
- Conditional statements
- Loops
- Functions
- Simple algorithms
- Problem-solving exercises

## 💻 Technologies Used

- Python 3
- GitHub for version control

## 🚀 Purpose of This Challenge

The purpose of this challenge is to develop **consistent coding habits** and gradually improve programming skills by practicing Python every day.
